# Golden Hour Express 🚂

A tiny experiment in passing audio data to shaders. A little train, hurtling through space, dancing to the music.

## About

Sometimes life feels like you're hurtling through space, unsure of the destination — but you keep going through it, and it's more fun when you can dance to the music. Hit play and watch the universe dance with you.

Built with React Three Fiber, custom GLSL shaders, and the Web Audio API. Bass swells the nebula, treble makes it sparkle, and the warp field drifts on regardless.

## Run it

```bash
npm install
npm run dev
```
